# Projeto 2 - Sistemas de Busca

Repositório destinado à implementação e análise estatística de algoritmos de busca (Sequencial, Binária e em Árvore Binária de Busca - BST), desenvolvido para a disciplina de Estruturas, Pesquisa e Ordenação de Dados.

## Tecnologias Utilizadas
* **Linguagem:** Java
* **Compilação e Execução:** JDK (Java Development Kit) nativo.

## Como compilar e executar o projeto

1. Certifique-se de ter o Java instalado em sua máquina.
2. Clone este repositório ou baixe os arquivos fonte (`.java`).
3. Abra o terminal na pasta onde os arquivos estão localizados.
4. Compile o código com o comando:
   javac *.java
   java main
