📚 Trabalho - Estrutura de Dados II

Este repositório contém o desenvolvimento do trabalho da disciplina de Algoritmos e Estrutura de Dados II, com foco na implementação, análise e comparação de diferentes estruturas e algoritmos.

🎯 Objetivo

O objetivo deste projeto é aplicar, na prática, conceitos fundamentais de estruturas de dados, analisando desempenho, eficiência e comportamento em diferentes cenários.

As estruturas de dados são essenciais para a organização e manipulação eficiente de informações, influenciando diretamente a performance de sistemas computacionais.

🧠 Conteúdos abordados

Este projeto pode envolver (dependendo da proposta do trabalho):

Estruturas de dados avançadas
Algoritmos de ordenação
Árvores (AVL, Rubro-Negra, etc.)
Heaps e filas de prioridade
Tabelas hash
Análise de desempenho (benchmark)

Esses temas fazem parte do conteúdo típico da disciplina, incluindo árvores, grafos, ordenação e estruturas avançadas.
